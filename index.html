Xbox Kinect Checkers

Checkers is a Chrome extension that blocks AdSense from determining your demographic info. It's run either passively or actively.

When run actively, a user selects a suggested demographic to add themselves to. Then, AdSensible performs relevant Google searches until the demographic is added to their profile. Under the hood, AdSensible takes the user's demographic, queries a Flask server which asks the DataMuse API for 100 similar words, then pipes the similar words into noun/verb search templates, which generates a search query. Ex.: "baking" --> "cooking club near me". We dealt with issues like rate-limiting, pinging ad-trackers without actually loading their page, and reverse-engineering AdSense.

I helped with the reverse-engineering and handled all the Chrome extension code. This involved passing data between background scripts, content_scripts, and browser_action scripts. I also designed the front end.

I built AdSensible in a team of three at HackRPI 2019, and it won 2nd Best Hack Overall, Best Data Privacy Hack, and a Wolfram Alpha award.
